## UV Fluorescence Spectrometer Code

Hello! This is the code I wrote for the UofT RSX 2026 Science team. This year, our project was to create a UV vis spectrometer. This repository contains everything fromt the notes to the code we used to drive the spectrometer head.


### Important Hardware Notes:
The code written for the UV spectrometer is based on our testing, there are a few ways it differs from the C12880MA official datasheet since ours had a defect. In any case, as far as I am aware, this is the only implementation of auto-exposure for the Hamamatsu C12880MA spectrometer head (cool I guess).
